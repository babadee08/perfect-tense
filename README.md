This is a very simple wrapper for the Perfect tense library. Mostly just converting the procedural code into OOP. Once is works fine, I/We would refactor into a proper library. The Original code contains some sample implementations of the original library.

## Installation

You can install the package via composer:

```bash
composer require composer require babadee08/perfecttense
```

## Usage

``` php
$skeleton = new Spatie\Skeleton();
echo $skeleton->echoPhrase('Hello, Spatie!');
```
